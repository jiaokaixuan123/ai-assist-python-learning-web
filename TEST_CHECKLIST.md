# 前端功能测试清单

## 1. 用户认证测试

### 注册功能
- [ ] 访问 http://localhost:5173/register
- [ ] 输入用户名（2-20字符）、密码（≥6位）、邮箱（可选）
- [ ] 点击"注册"按钮
- [ ] 验证：自动跳转到首页，右上角显示用户名

### 登录功能
- [ ] 访问 http://localhost:5173/login
- [ ] 输入已注册的用户名和密码
- [ ] 点击"登录"按钮
- [ ] 验证：跳转到首页，显示用户名和学习统计

### 登出功能
- [ ] 登录后点击右上角"退出"按钮
- [ ] 验证：用户名消失，统计卡片消失
